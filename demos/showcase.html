Get Started

The process

Three steps. One command.

Argo turns your existing Playwright tests into cinematic product demos — no screen-recording software required.

01 — WRITE

✎️

Write a demo script

Author your demo as a Playwright test using the familiar @argo-video/cli API. Add overlay markers and narration cues alongside your normal page interactions — no new tools to learn.

02 — RECORD

🎥

Record the browser

Argo launches a headless Chromium at 1920×1080 and captures your script frame-by-frame. Overlays are composited live into the recording — what you script is exactly what viewers see.

03 — EXPORT

💾

Export the video

AI narration is generated from your JSON manifest and mixed into the final MP4. One pipeline command handles recording, voiceover synthesis, and FFmpeg export — ready to ship.

Features

Everything you need.

Professional output without the production overhead. Argo handles the complexity so you can focus on the demo.

🆕

Overlays

Lower-thirds, headline cards, callouts, and image cards. Typed, timing-aware, auto-themed.

showOverlay()
🎤

Voiceover

AI narration from a JSON manifest. Six TTS engines — local or cloud. Scene pacing follows clip lengths.

{ "scene": "intro", "text": "Welcome." }
narration.mark()
🔍

Camera Effects

Spotlight, focus ring, dim-around, zoom-to. Direct attention without rewriting your app.

zoomTo()
⚙️

Pipeline

One command. TTS, record, align, export. Subtitles, chapters, and scene report included.

argo pipeline

Voice

Six engines. One config line.

Start local and free, then swap to a cloud voice for the final cut. All engines cache clips by content hash.

Kokoro

Local, free, zero API keys. Default engine for fast iteration.

engines.kokoro()

mlx-audio + Voice Clone

Clone your voice from a 15s clip. Local, private, Apple Silicon. No data leaves your machine.

refAudio: './my-voice.wav'

OpenAI

tts-1 and tts-1-hd models. Six voices including alloy, nova, and shimmer.

engines.openai()

ElevenLabs

Ultra-realistic voices with stability and similarity controls.

engines.elevenlabs()

Gemini

Google's Gemini TTS with multilingual support.

engines.gemini()

Sarvam

Indian language TTS — Hindi, Tamil, Telugu, and more.

engines.sarvam()

Camera Language

Direct the eye.

Four effects that layer cleanly with overlays. Non-blocking by default — fire-and-forget safe.

💡

Spotlight

Dims everything except the target. A dark overlay with a precision-cut hole draws all attention to one element.

spotlight(page, selector)
🔴

Focus Ring

A pulsing glow border on the target element. Subtler than spotlight — highlights without dimming the page.

focusRing(page, selector)
🌓

Dim Around

Fades sibling elements to 30% opacity. The target stays full brightness while everything else recedes.

dimAround(page, selector)
🔎

Zoom To

Smoothly scales and centers on a target element. Overlays stay fixed while the page content zooms.

zoomTo(page, selector)

Example

See the script.

A complete demo script — from import to overlay — in under fifteen lines.

demos/my-demo.ts 
import { test } from '@argo-video/cli';
import { showOverlay } from '@argo-video/cli';

test('my-demo', async ({ page, narration }) => {
  await page.goto('/');

  narration.mark('intro');
  await showOverlay(page, 'intro', {
    type: 'lower-third',
    text: 'Welcome to our app',
  }, 3000);
});
