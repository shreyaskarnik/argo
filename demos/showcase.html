Voiceover

Iterate with local TTS, switch to a premium engine for the final cut, or clone your own voice for branded walkthroughs. Scene-level voice, speed, and language are part of the same manifest.

Camera Language

Direct attention like an editor, not a screen recorder.

Camera helpers are how the canonical showcase stays watchable. Every new feature gets a focused beat instead of a full-page scroll-and-pray.

Effect

spotlight()

Darken everything except the thing you need the viewer to understand first.

Effect

focusRing()

Use a pulse instead of a full dim when you want a lighter touch.

Effect

dimAround()

Fade siblings away so the one changed card or command becomes obvious.

Effect

zoomTo()

Punch into a card or control when the current scene needs a tighter visual beat.

Effect

showConfetti()

Use sparingly for launches, wins, or the last scene of the showcase.

Effect

cursorHighlight()

Keep the eye on the pointer during dense walkthroughs without painting giant arrows on screen.

Export Stack

Fleet Workflow

When you have more than one demo, Argo scales with you.

The same tools that make a single showcase strong also help when the product has multiple journeys, environments, and owners.

Batch

argo pipeline --all

Run every demo in the configured demos directory and fail the batch when any important path breaks.

Dashboard

Preview every demo from one view

Status, resolution, quick preview links, and a clearer picture of what needs rerendering.

Validation

Catch bad scene wiring before a render

Validate scene names, imports, and manifest structure before you spend time generating narration or recording.

Doctor

Check the environment

ffmpeg, Playwright browsers, config, and the other failure modes that usually burn time right before a render.

Code

It is still just a Playwright script.

That is the real wedge. The video stays maintainable because the demo logic is still code your team already understands.

demos/showcase.demo.ts 
import { test, demoType } from '@argo-video/cli';
import { showOverlay, zoomTo, focusRing } from '@argo-video/cli';

test('showcase', async ({ page, narration }) => {
  await page.goto('/showcase.html');
  narration.mark('preview');
  await demoType(page, '#preview-text-field', 'Tighten the preview voice line.');
  void zoomTo(page, '#preview-export', { scale: 1.14, duration: 1800 });
  await focusRing(page, '#preview-export', { wait: true });
  await showOverlay(page, 'preview', narration.durationFor('preview'));
});
