Six new features in Argo

Scene transitions, speed ramp, GIF export, batch pipeline, multi-demo dashboard, and a progress bar.

At a Glance

Every feature ships in this release — no API keys, no cloud services, just config.

🎬

Scene Transitions

Fade through black, dissolve, or wipe between scenes. Configured per-project in your export settings.

⚡

Speed Ramp

Compress gaps between scenes at 2× or more. Navigation and page loads fly by; key interactions stay at full speed.

🖼️

GIF Export

Two-pass palette-optimized animated GIFs for docs, READMEs, and pull request previews.

📦

Batch Pipeline

Build every demo in one command. Continues on failure — one broken demo won't block the rest.

📊

Dashboard

See all your demos at a glance with build status, file sizes, and quick-action links.

📈

Progress Bar

Watch encoding progress in real time. No more staring at a silent terminal wondering if ffmpeg is still alive.
